# SCSS Normalize

A modern alternative to the classic normalize.css is a minimum of code, maximum benefit.

![npm](https://img.shields.io/npm/v/@m2collective/scss-normalize?style=for-the-badge)
![npm](https://img.shields.io/npm/dt/@m2collective/scss-normalize?style=for-the-badge)
___

## Installation

This package can be deployed automatically using NPM:

```
npm i @m2collective/scss-normalize
```

## Usage

To use it, import the package:

```scss
@use "@m2collective/scss-normalize";
```
